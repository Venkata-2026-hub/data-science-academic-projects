# Assignment 1 – Machine Learning Models

This project applies classical machine learning algorithms to financial datasets.

## Pipeline

1. Data preprocessing
2. Feature engineering
3. Model training
4. Model evaluation

## Models Implemented

- Linear Regression
- Polynomial Regression
- Lasso Regression
- Random Forest
- XGBoost

## Evaluation Metrics

- RMSE
- MAE
- R²

The results show that ensemble tree-based models provide the best predictive performance.
