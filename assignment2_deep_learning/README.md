# Assignment 2 – Deep Neural Network Modeling

This project implements deep learning models using TensorFlow/Keras in R.

## Pipeline

1. Environment setup
2. Data loading and cleaning
3. Feature preprocessing
4. Dataset splitting
5. Neural network architecture development
6. Hyperparameter tuning
7. Model evaluation

## Neural Network Experiments

- Multiple hidden layer architectures
- ReLU, ELU and Leaky ReLU activation functions
- K-fold cross-validation
- Oversampling for class imbalance

## Tools

- TensorFlow
- Keras
- tfruns
